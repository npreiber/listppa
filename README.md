# listppa

A zshell plugin to list all installed ppas for use with oh-my-zsh

## requires
zsh and ohmyzsh

## Installation
clone the repository in ~/.oh_my_zsh/custom/plugins

add listppa to plugins in zshrc:

plugins=(
	...
	listppa
	...
)

## use
type listppa in command line
